Chia code:
 - nhóm nào xong trước sẽ hỗ trợ nhóm còn lại code tiếp
 - Ai code chổ nào thì commen lại tên mình vào phương thức đó luôn
 - ( nếu muốn thì làm ) Đặt dòng System.out ("Người viết: "+ <tên>) vào đầu phương thức luôn cho dễ, nào xong xóa sau.
 - 1 method chỉ do một người code
 - xem code mẫu của tớ trước khi code (nếu có), không hiểu chổ nào thì hỏi tới, tại t viết khá nhiều hàm hỗ trợ rồi chỉ cần gọi chạy thôi


    Nhóm công việc 1: đọc ghi file
        - code đọc ghi dữ liệu
        - code ghi object vào file txt ở các phương thức saveDate ở các lớp có chữ Manager đứng đầu
        - code đọc object từ file txt ở các phương thức loadDate ở các lớp có chữ Manager đứng đầu
        - phần này đơn giản nhất. Làm rất nhanh. Làm xong người này sẽ thực hiện nhóm công việc 3 luôn

    Nhóm công việc 2: code các scene (tạo các menu lựa chọn, ràng buột các input của người dùng nhập,...)
        - chỉ ghi các hàm, nếu chửa thực hiện được thì in ra cho người thực hiện nhóm công việc 3 hoàn thành
        - chỉ code trên các Class có chữ Scene đứng đầu (Không code trong SceneManager vì t đã viết xong rồi chỉ cần lấy sử dung thôi)
        - xem code mẫu ở 2 Class SceneLogin và SceneMainMenuMember và code giống vậy

    Nhóm công việc 3: hoàn thành các phương thức ở các lớp còn lại
        - xem code mẫu ở 2 Class SceneLogin và SceneMainMenuMember để phối hợp với người làm nhóm công việc 2
        - ưu tiên làm những gì người ở nhóm 2 yêu cầu trước
        - tiếp tục làm 1 mình cho tới khi người nhóm 1 xong thì chia nhau làm tiếp.
